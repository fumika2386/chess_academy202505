## 制作サイトの説明（40文字程度）
　HTMLとCSS の基礎知識で作成したLPです。仕様書の再現を試みました。
## 工夫した点・こだわった点
　まずは仕様書通りにと作成しましたが、時間の見積もりを誤っており、焦りながら進めてしまいました。
　次回は可能な限り早めの課題着手を心がけ、自分なりの工夫までできるよう進めていきたいと思います。
## 難しかった点・次回トライしたこと（または機能）
　・2枚目のスクリーンショットですが、仕様書通りにと思いpタグで段落分けしていたのですが、テキスト１６pxにするとこのような見栄えになってしまい、画面収録はテキスト１４pxに変更して行いました。そのように、仕様書通り忠実に再現できなかった部分が何ヶ所かあります。
　・フォームのHTMLもCSSも難しかったです。検索するにも、なんて検索していいのか分からず、チャットGPTにニュアンスだけ伝えて、検索力の部分から補ってもらっていました。「フォームの右側の選択肢選べるやつって名前何？」など聞いてました、、。
